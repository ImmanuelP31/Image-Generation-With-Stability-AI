A Python desktop application that generates AI images from user prompts using Stability AI's Stable Diffusion API. Built with CustomTkinter for a neat and attractive GUI.

## Features

- Enter text prompts and choose image styles
- Generates 512x512 images using Stability AI's API
- Styles include: Realistic, Cartoon, 3D Illustration, and Flat Art
- Attractive GUI with CustomTkinter

## Getting Started

### Prerequisites

- Python 3.8+
- API key from [Stability AI](https://platform.stability.ai)

### Install dependencies

```bash
pip install -r requirements.txt
